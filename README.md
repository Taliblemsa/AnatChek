🏫 ระบบเช็คชื่อ ส.ธ. (S.T. Attendance System)

ระบบเช็คชื่อนักเรียนออนไลน์ที่ออกแบบมาเพื่อความรวดเร็วและรองรับกิจกรรมที่หลากหลายภายในโรงเรียน

🚀 ฟีเจอร์หลัก

เลือกชั้นเรียนได้ง่าย: รองรับการเช็คชื่อ ม.2, ม.4, ม.5, ม.6

กิจกรรมพิเศษ: รองรับการเช็คชื่อเข้าแถวหรือกิจกรรมเช้า เช่น "อัลกุรอาน(ตอนเช้า)"

Responsive Design: ใช้งานได้ดีทั้งบนมือถือ แท็บเล็ต และคอมพิวเตอร์

Fast Integration: เชื่อมต่อกับ Google Sheets ผ่าน Google Apps Script

📸 ภาพตัวอย่างระบบ

(ใส่รูปภาพ Screen ของแอปที่นี่)

🛠 เทคโนโลยีที่ใช้

Frontend: HTML5, Tailwind CSS

Backend: Google Apps Script

📝 วิธีการใช้งาน

เลือก ระดับชั้น หรือ กิจกรรม ที่ต้องการจากเมนู

ระบบจะแสดงรายการห้องเรียนที่เกี่ยวข้อง

กดปุ่ม "เข้าสู่ระบบเช็คชื่อ" เพื่อบันทึกข้อมูลเข้าสู่ฐานข้อมูล
